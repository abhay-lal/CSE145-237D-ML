TinyML Owl Vocalization Classifier

On‐device detection and classification of burrowing‐owl calls using a TinyML pipeline on STM32H7.

Overview Resources Contact

Project Overview

This project implements a lightweight TinyML pipeline to process burrowing‐owl audio recordings, convert them to Mel spectrograms, train compact neural networks (MobileNetV2 & ProxylessNAS), quantize to int8, and deploy the final model on an STM32H7 microcontroller. Key features:

  • 64‐band Mel spectrogram preprocessing (Python + Torchaudio)
  • Model training on BUOWSET dataset with 6 vocalization classes
  • Quantization (PyTorch→ONNX→TensorFlow→TFLite int8) and C‐header generation
  • On‐chip DSP (KissFFT/CMSIS‐DSP), inference loop, and power‐saving sleep/wake logic

For full setup instructions, folder structure, and detailed steps, please see the project’s README.
